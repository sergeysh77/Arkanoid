This is a classic Arkanoid (Breakout) game implementation for Windows using Win32 API. The player controls a paddle at the bottom of the screen to bounce a ball and destroy bricks arranged at the top. The game features:

- 3 brick types: Normal (1 hit), Hard (2 hits), Strong (3 hits)
- 5 powerups: Enlarge paddle, Shrink paddle, Slow ball, Sticky ball, Extra life
- 2 languages: English and Russian
- Sound effects embedded as resources
